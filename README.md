# Life OS

A personal Life Operating System — a modular, version-controlled repository for managing the systems of a life: tasks, time, knowledge, goals, and routines.

## Status

**v0.1 — Bootstrapping.** The repository currently contains only its foundational documents. Modules are added one at a time, each requiring explicit approval before work begins.

## Mission

Build a Life Operating System that compounds for decades by aligning identity, systems, execution, health, knowledge, and AI into one integrated architecture.

## How this repository is built

This repo is implemented incrementally from `Life_OS_Blueprint_v1.pdf`, an architecture document maintained outside the repository. That blueprint is the source of truth — this repo implements it, it does not redesign it. Each module is built one at a time, in the blueprint's defined build order, never all at once.

- [CLAUDE.md](CLAUDE.md) — working rules for how this repository is built
- [ROADMAP.md](ROADMAP.md) — what's planned, what's next
- [CHANGELOG.md](CHANGELOG.md) — what's shipped, in order

## Principles

- **Modular.** Each part of the system stands on its own and can be understood in isolation.
- **Cross-linked.** Related files reference each other so the repository can be navigated like a graph, not just a tree.
- **Markdown-first.** Plain text, readable without tooling, durable over time.
